fbooks
======

読書記録

# 課題
読書記録を管理するアプリケーションを作成してください。  
レイアウトや画面構成はお任せします。

# 目的
1. Java を修得する。
2. グループによる開発を経験する。
 
# 開発環境
1. 言語：Java 8, JavaFx 8
2. IDE：Pleiades All in One eclipse Kepler
3. バージョン管理：github
4. データベース：Sqlite

# 要件
1. 本を登録できる。
 2. タイトル、著者、出版社を登録できる。
 1. 読んだ期間を記録できる。
 2. 感想やメモを記録できる。
1. 本を検索できる。
 1. 読んだ期間を指定できる。
 2. 読んだジャンルを指定できる。
 3. 著者を指定できる。
